
# n8n RAG Agent

## Features
- Upload documents
- Vector database
- Retrieval-Augmented Generation (RAG)
- LLM integration
- Chat interface

## Tech Stack
- n8n
- OpenAI/Groq/Gemini
- Qdrant/Pinecone/Chroma
- Embeddings Model

## Import

Download `workflows/rag-agent.json` and import it into n8n.

## Credentials Required

- OpenAI API Key
- Vector Database API Key
- Any other credentials used
